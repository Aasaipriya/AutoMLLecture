    \item Generic:
    \begin{itemize}
      \item Polynomial Features
      \item Normalize 
      \item PCA
    \end{itemize}
    \item Heuristisc
    \begin{itemize}
      \item Detecting Dates, Hours: Transofrm to "circular" features within year, month, day, or whatever periodicity we assume \\

    \end{itemize}

    \begin{itemize}
      \item Names $\leftarrow$ gender, ethnicity, age
